Parameter Value Units Type
Flowrate m³/h input
Total dynamic head m input
Fluid density kg/m³ input
Pump efficiency % input
Shaft power required kW calculation

Power is approximated as \( P\,[\text{kW}] \approx \frac{Q\,[\text{m³/h}] \times H\,[\text{m}] \times \rho\,[\text{kg/m³}] \times 9.81}{3.6 \times 10^6 \times \eta} \), with \(\eta\) as decimal efficiency.[web:59]

NPSH Check (Available)

Parameter Value Units Type
Absolute pressure on liquid surface bar(a) input
Static suction head (positive above pump) m input
Suction line friction losses m input
Fluid vapor pressure bar(a) input
NPSH available m calculation
Pump NPSH required (from curve) m input
NPSH margin (NPSHA − NPSHR) m calculation

NPSH available is approximated as \( \text{NPSHA} = h_1 + h_{ss} - h_f - h_{vp} \) with \(h_1\) and \(h_{vp}\) converted from absolute pressure (bar) to head (m).[web:58][web:60]
