How to add new sizing tools

To add a new equipment sizing page, create a new HTML file (e.g. thickener_sizing_tool.html) and register it in the configuration in the script below.

  • Create the new sizing page file in this folder (copy an existing tool as a starting template).
  • Add a new entry to the equipmentPages object with a label, description, and file name.
  • The portal will automatically show a new button for that equipment type.
